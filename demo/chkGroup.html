Installation

  1. Copy the necessary file(s) to your project (directives/ChkGroup)
  2. Change demoApp reference in directives/ChkGroup.js to your module reference.
  3. Add directive in template, check for parameters in source code/chkGroup directive

Demo

No initial selection

Code

<chkgroup selected-value="selectedOptions"
          options="demoCollection"
          id-field="id"
          label-field="label">
</chkgroup>

Result

Currently selected: {{selectedOptions}}

Second and third option already selected

Code

<chkgroup selected-value="selectedOptions2"
           options="demoCollection"
           id-field="id"
           label-field="label">
</chkgroup>

Result

Currently selected: {{selectedOptions2}}
